Important Notes & Troubleshooting

Things to Note

Temporary File Size

Because the software captures uncompressed data initially, the temporary .screc files can be quite large. Ensure you have adequate storage space. Once converted to MP4, the file size will be significantly reduced.

Drive Speed

For smoother 60 FPS recording, a storage drive with decent write speeds (like an SSD) is recommended to avoid dropped frames.
